NOTEABLE BUGS:

- Logged Food located in the calculator is too spaced out.
- Attempting to delete an item off the logged food list will cause the app to crash.

Additional information:

- App lacks features presented on the web app.
- App lacks restaurant data
- App does NOT run off a database.

SETUP:

On Visual Studio Code, open the bigtrack project folder.

Open a new terminal and install the Kivy libraries:

pip install kivy[full]

Run the app using

python main.py
